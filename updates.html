üíæ Message Persistence with Local Storage

Messages are now cached in browser's local storage to prevent complete loss during server restarts:

  • Last 200 messages are stored in browser's local storage
  • Messages persist even if server restarts
  • When reconnecting, messages are restored from local storage
  • New messages from server replace the oldest cached messages
  • Storage is automatically cleared if unused for 30 days
  • Users can manually clear chat history from settings

üêû Fixed Issues:
